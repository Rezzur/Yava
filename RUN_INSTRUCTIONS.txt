ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРОЕКТА YAVIMAX

1. ПРЕДВАРИТЕЛЬНЫЕ ТРЕБОВАНИЯ
- Установленный Docker Desktop
- Java 21 (проект настроен на работу с JDK 21+)
- Node.js (версия 18 или новее)

2. ЗАПУСК ИНФРАСТРУКТУРЫ (БАЗА ДАННЫХ И REDIS)
Перейдите в папку backend и запустите контейнеры:
   cd backend
   docker-compose up -d

Это поднимет:
- PostgreSQL на порту 5432
- Redis на порту 6379

3. ЗАПУСК БЭКЕНДА (SPRING BOOT)
Из папки backend выполните команду:
   .\gradlew.bat bootRun

Сервер запустится на порту 8080.
Swagger документация будет доступна по адресу: http://localhost:8080/swagger-ui.html

4. ЗАПУСК ФРОНТЕНДА (VITE + REACT)
Перейдите в папку frontend, установите зависимости (если еще не установлены) и запустите режим разработки:
   cd frontend
   npm install
   npm run dev

Приложение обычно запускается на http://localhost:5173
Если порт 5173 занят, Vite может предложить http://localhost:5174

5. АВТОРИЗАЦИЯ И РЕГИСТРАЦИЯ
- Страница входа: http://localhost:5173/login
- Если регистрация выдает ошибку "Registration failed" без деталей, скорее всего, пользователь с таким Username или телефоном уже существует. Попробуйте войти под своими данными.

6. ДОПОЛНИТЕЛЬНЫЙ МОДУЛЬ (FRONTEND-WEB)
Если вам нужно запустить вторую версию фронтенда:
   cd frontend-web
   npm install
   npm run dev
